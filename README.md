# Advanced AI
# PERCI

AI kiosk project using YOLO for person detection and Whisper for speech transcription.

## Requirements
- Python 3.10 or 3.11 recommended
- Webcam
- Microphone
- Windows PowerShell for text-to-speech
- FFmpeg installed and added to PATH

## Setup

### Windows
Clone the repo, open the folder in terminal, then run:

```bat
setup.bat
```
